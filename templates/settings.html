Technical Standards
Stack: Python / Flask · SQLite on Railway /data volumes · Bootstrap 5 · Jinja2 templates

Non-negotiable rules:
  • All password fields have show/hide eye toggle (👁️/🙈)
  • CSRF tokens on all forms
  • Security headers on all routes
  • No debug=True in production — ever
  • /health endpoint on every app returning JSON {"status":"ok"}
  • Rate limiting on login + API endpoints
  • bcrypt for all passwords
  • Push to GitHub after every meaningful change with a descriptive commit message
  • GitLab is backup — push to both after significant changes
