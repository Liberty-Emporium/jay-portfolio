📦 How to add monitoring to any app
  1. 1.Copy echo_reporter.py into the app's root folder (same level as app.py)
  2. 2.Add 2 lines to the app's app.py — right after app = Flask(__name__):
  3. 3.Set 2 env vars in Railway for that app service
from echo_reporter import install_reporter
install_reporter(app, 'Your App Name Here')
Railway env vars to add to each app: 
ECDASH_REPORTER_URL   = https://jay-portfolio-production.up.railway.app
ECDASH_REPORTER_TOKEN = (your shared token — same for all apps)
Also add GITHUB_TOKEN and ECDASH_REPORTER_TOKEN to this EcDash service so it can write Echo's memory to GitHub.
